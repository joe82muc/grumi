# NT Feedback App (App2)

Starten:
1. Terminal im Ordner App2 öffnen
2. `node server.js` oder `npm start`
3. Im Browser öffnen: `http://localhost:3000/feedback.html`

Seiten:
- `feedback.html` = Eingabe durch Schüler
- `auswertung.html` = Live-Auswertung

Hinweise:
- Codes stehen in `data/tokens.json` (beim ersten Start automatisch erzeugt)
- Jede Person kann jede Fremdgruppe genau einmal bewerten
- Eigene Gruppe ist gesperrt
